## Quick Reference

### Starting a Session
```
/start-session [project-name]
```
Loads context from PROJECT.md and shows last session summary.

### Ending a Session
```
/end-session
```
Captures completed work, decisions, and sets next priorities.

### Mid-Session Logging
```
/iterate [description]
```
Logs significant attempts with results and learnings.
