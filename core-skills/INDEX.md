# Core Skills Index (DEPRECATED)

> **This directory is deprecated as of v2.3.**
>
> Skills have been migrated to `.claude/skills/` using Anthropic's official v2.0+ format.

---

## New Location

All skills are now in: **`.claude/skills/*.md`**

Skills use YAML frontmatter for auto-activation:

```markdown
---
name: skill-name
description: "Brief description for auto-activation"
---

# Skill Content...
```
